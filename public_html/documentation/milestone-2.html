Colors should be monochromatic blue. Font should be modern and clean. It should be very easy to read. There should be enough whitespace around headers for customers to read quickly.

The header should have a logo on it. The links should move to the specific parts of the page that they reference.

Jumbotron:

Jumbotron has a graphic that has shows a price tag and some appliances. Use H1 DataTags as the header. The smaller text should use "Making Price Tags Easy". Perhaps a paragraph with a few lines obout the service if it doesn't make the space too busy. Not sure we need a call to action button unless we can make it go to the sign up form at the bottom.

Benefits:

Pricing:

Headers will feature product names. Brief tag line or benefit underneath. Pricing will be in big bold numbers.
